# 🎲 Multiplayer Ludo Game

A real-time multiplayer Ludo game built with **React**, **Node.js**, **Socket.IO**, **Redis**, and **PostgreSQL**.

Players can create a room, share the room ID with friends, and play together in real time. The game also supports reconnecting to an ongoing match, allowing players to rejoin if they get disconnected.

## ✨ Features

* 🎮 Create a game room
* 🔗 Share Room ID with friends
* 🚪 Join a room using Room ID
* 👑 Room owner can start the game
* ⚡ Real-time gameplay using Socket.IO
* 🔄 Rejoin an active game after disconnection
* 💾 Game state management with Redis
* 📱 Responsive and modern UI

## 🎯 How to Play

1. Create a room.
2. Share the generated Room ID with your friends.
3. Friends join the room using the Room ID.
4. The room owner starts the game.
5. Enjoy real-time multiplayer gameplay.
6. If a player disconnects, they can rejoin the ongoing game without losing their progress.
